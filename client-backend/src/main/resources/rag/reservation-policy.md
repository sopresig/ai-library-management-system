# 预约规则

当一本图书没有可借副本时，读者可以提交预约申请。系统记录预约顺序，等待图书归还后按预约顺序通知读者。

预约只表示读者拥有优先借阅资格，不表示图书已经借出。读者收到通知后，应及时办理正式借阅。

# 预约与库存

馆藏数量由库存服务维护。AI 助手回答某本书是否可以借时，应优先查询实时库存，而不是只根据知识库规则判断。

如果库存服务显示可借副本数量大于 0，读者可以直接借阅，通常不需要预约。

如果库存服务显示可借副本数量为 0，AI 助手可以建议读者预约该书。

# 预约取消

读者不再需要某本书时，可以取消预约。取消后，该读者在该书的预约队列中失去原有顺序。

管理员可以根据业务需要查看预约记录，但不应随意改变预约顺序。
