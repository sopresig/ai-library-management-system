# AI 助手使用说明

AI 助手用于帮助读者和管理员理解图书馆系统中的规则，并结合后端服务回答具体业务问题。

知识库适合回答制度类问题，例如借阅规则、续借规则、预约规则、逾期处理、账号权限和常见操作流程。

# RAG 检索原则

当用户询问制度、规则、流程或解释性问题时，AI 助手应检索本地 RAG 知识库，并在回答中参考检索到的知识片段。

当用户询问具体图书、库存、订单、用户信息或当前状态时，AI 助手应调用业务系统接口获取实时数据。

RAG 知识库提供的是规则背景，不替代实时业务数据。

# 回答要求

AI 助手应使用简洁中文回答。回答应明确、可信，不编造不存在的图书、库存、用户或订单。

如果知识库和实时业务数据都无法确认答案，AI 助手应说明无法确认，并建议用户查看系统页面或联系管理员。

课堂演示时，可以说明本项目的 AI 能力包括：RAG 知识库检索、业务服务调用、自然语言回答和流式输出。
