# 借阅规则

普通读者每次最多可以借阅 5 本图书，单本图书的默认借期为 30 天。系统创建借阅订单时，会同时记录读者、图书、借出日期、应还日期和订单状态。

管理员可以在后台查看读者当前借阅记录、历史订单和逾期状态。读者在客户端发起借阅请求后，系统会检查库存服务中的可借副本数量。

如果图书存在可借副本，系统会生成借阅订单，并减少对应图书的可借库存。如果图书没有可借副本，读者应使用预约功能，而不是重复提交借阅申请。

# 借阅限制

读者存在逾期未还图书时，系统可以限制其继续借阅新的图书，直到逾期图书完成归还。

读者的借阅数量达到上限时，系统应拒绝新的借阅请求，并提示读者先归还部分图书。

知识库只保存通用借阅规则。某一本书当前是否可以借阅，应优先调用库存服务查询实时馆藏数量。
