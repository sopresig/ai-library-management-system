# 用户账号与权限

系统区分普通读者和管理员。普通读者主要使用客户端完成图书搜索、借阅、预约、续借和查看订单。

管理员主要使用后台功能管理图书、分类、库存、用户和借阅订单。

# 登录与认证

用户需要登录后才能执行与个人账号相关的操作，例如提交借阅申请、查看个人借阅记录、续借图书和取消预约。

如果用户未登录，系统应提示用户先登录。AI 助手不应要求用户提供密码，也不应在回答中泄露认证信息。

# 权限边界

普通读者不能直接修改图书库存、删除图书、修改分类或管理其他用户。

管理员操作应通过后台接口完成，并由系统进行权限校验。

AI 助手回答权限问题时，应区分“读者可以做什么”和“管理员可以做什么”。
