# 图书馆常见问题

读者可以在客户端搜索图书、查看图书详情、查看库存、提交借阅申请、查看借阅历史和归还状态。

管理员可以在后台管理图书、分类、库存、用户和借阅订单。管理员操作应以系统真实数据为准。

# 图书搜索

读者可以按图书名称、作者、分类等条件搜索图书。搜索结果只说明图书资料存在，不一定表示当前有可借副本。

AI 助手回答“有没有某本书”时，可以结合图书服务查询图书资料；回答“能不能借”时，应进一步结合库存服务判断。

# 读者借阅状态

读者可以查看当前借阅中的图书、历史借阅订单、应还日期和逾期状态。

如果读者询问自己的具体订单、账号或借阅记录，AI 助手应调用业务系统接口，而不是只回答知识库中的通用规则。

# 回答边界

AI 助手不能编造不存在的图书、库存、用户或订单。知识库用于解释规则，业务数据应来自后端服务。
